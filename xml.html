XML

Extensible Markup Language (XML) är ett markeringsspråk (datautbytesformat) som definierar en uppsättning regler för kodning av dokument i ett format som är både mänskligt läsbart och maskinläsbart. World Wide Web Consortiums XML 1.0-specifikation från 1998 och flera andra relaterade specifikationer, alla av dem med öppna standarder, definierar XML. XML-designmålen betonar enkelhet, allmänhet och användbarhet på Internet. Det är ett textdataformat med starkt stöd via Unicode för olika mänskliga språk. Även om utformningen av XML fokuserar på dokument, används språket i stor utsträckning för att representera godtyckliga datastrukturer som de som används i webbtjänster. Det finns flera schemasystem för att hjälpa till att definiera XML-baserade språk, medan programmerare har utvecklat många applikationer programmeringsgränssnitt (API: er) för att underlätta behandlingen av XML-data.
